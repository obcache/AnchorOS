# Terminology
- Canonical namespace is **Anchor**.  
- “Capsule” replaces “jewel” in formal docs.  
- “Marker” (poetic alias: tombstone) for non-inflating references.  
- Tools should honor `legacy_terms` inside headers for backward search.

- Wherever “jewel” appears, read/write as **anchor/capsule**.
- All new artifacts must include `class: anchor/<subclass>` in YAML.
- Filenames should be **stable**; timestamps live in `timestamp`.
- Index entries add `class` column.
